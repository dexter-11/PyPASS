# PyPASS Desktop Password Manager v1

### Read about this Project [ here ](https://github.com/dexter-11/PyPASS/blob/main/About%20PyPASS.pdf)

### Installation:
Python > 3.7 required.

```sh
pip install -r reqs.txt
python pypass.py
```

## Features:
- Very less requirements
- User-friendly GUI
- Offline
- AES 256 Encrypted database
